Code Examples

copy paste and go

cURL 
curl https://anymousxeapi.vercel.app/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer any-YOUR_KEY_HERE" \
  -d '{
    "model": "gpt-5.4",
    "messages": [
      {"role": "user", "content": "whats good"}
    ]
  }'
Python (openai lib) 
from openai import OpenAI

client = OpenAI(
    api_key="any-YOUR_KEY_HERE",
    base_url="https://anymousxeapi.vercel.app/v1"
)

response = client.chat.completions.create(
    model="gpt-5.4",
    messages=[
        {"role": "user", "content": "whats good"}
    ]
)

print(response.choices[0].message.content)
JavaScript (fetch) 
const response = await fetch(
  "https://anymousxeapi.vercel.app/v1/chat/completions",
  {
    method: "POST",
    headers: {
      "Content-Type": "application/json",
      "Authorization": "Bearer any-YOUR_KEY_HERE"
    },
    body: JSON.stringify({
      model: "gpt-5.4",
      messages: [
        { role: "user", content: "whats good" }
      ]
    })
  }
);

const data = await response.json();
console.log(data.choices[0].message.content);

API Reference

everything you need to know in one place

POST

/v1/chat/completions

Send messages, get responses. Supports streaming. The main event.

model required

One of: gpt-5.4, gemini-3.1-pro-preview, glm-5

messages required

Array of message objects with role and content

stream optional

Set to true for streaming responses (SSE)

temperature optional

0-2, default 0.7. Higher = more creative

max_tokens optional

Max tokens to generate. Default 4096

GET

/v1/models

List all available models. No auth needed for this one.

Error Codes

401 Invalid or missing API key
400 Bad request — missing model or messages
429 Rate limited — chill for a sec
405 Wrong HTTP method
