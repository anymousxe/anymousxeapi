Code Examples

copy, paste, ship

cURL 
curl https://anylm.anymousxe-info.workers.dev/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer any-YOUR_KEY_HERE" \
  -d '{
    "model": "gpt-5.4",
    "messages": [
      {"role": "user", "content": "hello"}
    ]
  }'
Python 
from openai import OpenAI

client = OpenAI(
    api_key="any-YOUR_KEY_HERE",
    base_url="https://anylm.anymousxe-info.workers.dev/v1"
)

response = client.chat.completions.create(
    model="gpt-5.4",
    messages=[
        {"role": "user", "content": "hello"}
    ]
)

print(response.choices[0].message.content)
JavaScript 
const response = await fetch(
  "https://anylm.anymousxe-info.workers.dev/v1/chat/completions",
  {
    method: "POST",
    headers: {
      "Content-Type": "application/json",
      "Authorization": "Bearer any-YOUR_KEY_HERE"
    },
    body: JSON.stringify({
      model: "gpt-5.4",
      messages: [
        { role: "user", content: "hello" }
      ]
    })
  }
);

const data = await response.json();
console.log(data.choices[0].message.content);

API Reference

endpoints and parameters

POST /v1/chat/completions

Send messages and get completions. Supports streaming.

model required

Any model from the models list (e.g. gpt-5.4, claude-sonnet-4.6)

messages required

Array of message objects with role and content

stream optional

Set to true for streaming responses

temperature optional

0 to 2, default 0.7

max_tokens optional

Max tokens to generate. Default 4096

POST /v1/images/generations

Generate images with DreamShaper or P-Image. Requires Plus plan.

GET /v1/models

List all available models with pricing and tier info.

GET /v1/credits

Check your credit balance and recent transactions.

Error Codes

401 Invalid or missing API key
402 Insufficient credits
403 Plan upgrade required for this model
429 Rate limited, try again shortly
